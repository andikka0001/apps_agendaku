Panduan Penggunaan Aplikasi

Pendahuluan

Aplikasi Jadwal ini dirancang untuk membantu Anda mengatur kegiatan harian, jadwal kuliah/sekolah, dan catatan pribadi. Berikut adalah panduan lengkap penggunaan aplikasi.

Memulai Aplikasi

  • Masukkan Nama: Pada halaman awal, masukkan nama Anda di kolom yang tersedia.
  • NIM/Kelas (Opsional): Anda dapat memasukkan identitas tambahan jika diperlukan.
  • Tombol Masuk: Tekan tombol "Masuk" untuk mengakses menu utama aplikasi.

Menu Utama

  • Jadwal Kegiatan: Atur aktivitas harian Anda.
  • Jadwal Kuliah/Sekolah: Kelola jadwal akademik Anda per hari.
  • Ganti Tema Aplikasi: Ubah tampilan latar belakang aplikasi sesuai preferensi.
  • Buat Catatan: Simpan ide, tugas, atau informasi penting lainnya.
  • Info: Akses informasi pengguna dan panduan aplikasi.
  • Tentang Agendaku: Lihat informasi branding tentang aplikasi Agendaku.

Mengelola Jadwal

  • Edit Jadwal: Tekan tombol "Edit" di jadwal yang ingin Anda ubah. Anda akan dibawa ke halaman pengeditan khusus. Setelah selesai, klik "Simpan".
  • Reset Jadwal: Gunakan tombol "Reset" di halaman jadwal utama untuk mengembalikan jadwal ke pengaturan awal (default).
  • Format Penulisan (Jadwal Kegiatan): Pada halaman edit jadwal kegiatan, Anda akan menemukan opsi waktu dan kolom input. Pilih waktu yang sesuai (Shubuh, Pagi, Dhuhur, Siang, Ashar, Sore, Maghrib, Isya, Malam) lalu ketik kegiatan Anda. Contoh: pilih "Pagi" lalu ketik "makan mie".
  • Format Penulisan (Jadwal Kuliah/Sekolah): Pada halaman edit jadwal kuliah/sekolah, Anda akan menemukan input jam (misal 08:00 - 10:00) dan kolom input untuk mata pelajaran/deskripsi.

Tombol Bantuan

  • Tombol "💡": Tampilkan ringkasan jadwal hari ini.
  • Tombol "?": Akses bantuan kontekstual sesuai halaman yang sedang dibuka.
  • Tombol Rumah: Kembali ke halaman login.
  • Tombol Kutipan: Tampilkan motivasi harian.

Fitur Lanjutan

  • Penyimpanan Otomatis: Semua perubahan disimpan secara otomatis di perangkat Anda.
  • Notifikasi: Aplikasi akan memberikan notifikasi suara pada setiap interaksi penting.
  • Kustomisasi Tema: Pilih dari berbagai latar belakang yang tersedia.

Troubleshooting

  • Reset Aplikasi: Jika mengalami masalah, coba coba keluar dan masuk kembali.
  • Penyimpanan Penuh: Pastikan perangkat memiliki ruang penyimpanan yang cukup.
  • Kritik dan Saran: Gunakan fitur ini untuk melaporkan bug atau memberikan masukan kepada pengembang.
